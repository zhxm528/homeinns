# 私域指标规则文件

Q1私域指标考核产品 PRD

一、背景与目标

1.1 背景

集团在 2026 年将私域能力（企业微信、会员体系、小程序预订）作为核心增长引擎，需要将 私域拉新 → 会员沉淀 → 预订转化 → 高价值会员复购 的链路量化为可考核、可对比、可落地的指标体系，用于区域与门店的季度考核。

1.2 产品目标

将私域运营目标拆解为 4 个标准化指标

明确定义 口径、计算方式、考核边界、数据来源

支撑 区域 / 门店 Q1 考核与横向对比

指导一线执行动作，避免“只看结果、不知怎么做”

1.3 考核周期

周期：2026 年 Q1

对象：集团 / 区域 / 门店

指标一	企微每百人成功拉新	私域入口建立	35

三、指标一：企微每百人成功拉新

3.1 指标定义

衡量门店在 住店期间或离店当日，将可转化住客成功添加为门店企业微信的能力。

3.2 考核对象

门店

区域

3.3 指标口径

企微每百人成功拉新 =（成功拉新企微人数 ÷ 可拉新企微总人数）× 100

3.4 人群定义

可拉新企微总人数：

门店所有入住客人

剔除：非身份证入住客人、已添加任意门店企微的入住客人

成功拉新企微人数：

在考核有效期内，成功添加门店企微的入住客人

3.5 考核有效期

入住期间

或离店当天

3.6 Q1目标

集团 / 区域：35

四、指标二：逸粉每百人成功拉新

4.1 指标定义

衡量门店通过企微等方式，将可转化住客成功注册为逸粉会员的能力。

4.2 指标口径

逸粉每百人成功拉新 =（成功拉新逸粉人数 ÷ 可拉新逸粉总人数）× 100

4.3 人群定义

可拉新逸粉总人数：

门店所有入住客人

剔除：非身份证入住客人、已是逸粉会员的客人

成功拉新逸粉人数：

在考核有效期内完成逸粉注册的入住客人

4.4 考核有效期

入住期间

或离店当天

4.5 Q1目标

集团 / 区域：28

五、指标三：逸扉小程序渠道间夜占比

5.1 指标定义

衡量门店将预订行为从 OTA / 协议渠道迁移至 逸扉小程序 的能力。

5.2 指标口径

逸扉小程序渠道间夜占比 =（P3 逸扉小程序渠道预订间夜数 ÷ P3 全渠道预订间夜数）× 100%

5.3 数据范围说明

包含：有效完成的预订

不包含：退款、取消订单、非身份证预订

5.4 Q1集团参考值

平均目标：17%

区域差异化目标（示例）：

华南二区：23%

华北 / 上海：20%

六、指标四：金牛及以上逸粉本人预订且入住占比

6.1 指标定义

衡量 高等级会员（≥金牛） 是否形成稳定的 本人小程序预订 + 实际入住 行为。

6.2 指标口径

金牛及以上逸粉本人入住占比 =（P3 金牛及以上本人预订并入住的逸粉人数 ÷ P3 全渠道预订人数）× 100

6.3 人群定义

分子：

金牛 / 黑金 / 钻石逸粉

本人账号预订

实际入住

分母：

P3 全渠道预订人数

剔除非身份证预订

6.4 考核渠道

仅限：逸扉小程序渠道

6.5 Q1目标

集团平均：10%

七、一体化转化路径设计

运营阶段	核心动作	驱动指标	关键话术 / 策略
入住办理	引导添加企微	指标一	添加逸扉小助手，享专属服务与权益
入住办理	引导注册逸粉、实名认证	指标一、四	注册即享会员价，离店可升级金牛
住中服务	引导逸粉注册	指标二	小程序享智能服务、积分权益
预订引导	推送小程序专属优惠	指标二、三	小程序预订最低价、最高85折
离店引导	再次添加企微	指标一	添加企微，快速开票
离店关怀	企微回访、评价引导	指标三	入住评价有礼，复购优惠
