#实现一个表格的展示
##提示词输入参数：
- 输入查询条件
- 输入表格列
- 使用Ant Design组件
##页面布局
- 上方显示查询条件
- 中间显示已选择的查询条件
- 下方显示表格
##翻页规则
- 每页显示10条
- 可以更改每页显示条目：每页10条、每页50条、每页1000条
- 使用Ant Design翻页组件
##表格锁定规则
- 表格的第一行锁定
- 表格的第一列锁定
##查询条件的规则
- 查询条件是文本框或下拉框的都需要支持自查的模糊查询
- 查询条件支持重置按钮功能
- 查询条件按照每行3列展示
##表格数据显示规则
- 表格中是数字的，需要显示千分位，保留2位小数，小数末尾为0的不显示0
- 表格数据的第1行（整个表格的第2行）是所有记录的合计
##组件的规则
- 多选和单选的下拉框支持为空
- 使用Ant Design组件
