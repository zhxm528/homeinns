# 实现一个表格的展示

## 提示词输入参数
- 输入查询条件
- 输入表格列
- 使用 Ant Design 组件

## 页面布局
- 上方显示查询条件
- 中间显示已选择的查询条件，删除已选择的查询条件需要同步清空查询条件
- 下方用表格形式显示查询结果

## 翻页规则
- 每页显示 10 条（默认）
- 可以更改每页显示条目：每页 10 条、每页 100 条、全部
- 使用 Ant Design 翻页组件

## 表格锁定规则
- 表格的第一行锁定（表头固定）
- 表格的第一列锁定（左侧固定列）

## 查询条件的规则
- 查询条件是文本框或下拉框的都需要支持自查的模糊查询
- 查询条件支持重置按钮功能
- 查询条件按照每行 3 列展示

## 表格数据显示规则
- 表格中是数字的，需要显示千分位，保留 2 位小数，小数末尾为 0 的不显示 0

## 组件的规则
- 多选和单选的下拉框支持为空
- 使用 Ant Design 组件
